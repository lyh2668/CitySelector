# CitySelector
### 基于vue的移动端城市选择组件</br>
 ![二维码，显示失败请直接用链接访问](https://github.com/jwentao/CitySelector/blob/master/img/CitySelector.png)
 手机扫这里看demo
 <br/>
 针对移动端的城市选择组件<br/>
 可以使用拼音或关键字进行检索，也可以在右侧检索栏快速滑动</br>
 ### 本地构建方法：
 `git clone https://github.com/jwentao/CitySelector.git`   
 `cd CitySelector`   
 `npm install`  
 `npm run dev`<br/>
